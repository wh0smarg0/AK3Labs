# Workshop #3: Аналого-цифровий перетворювач (ADC) на STM32

## Опис проєкту
Цей проєкт демонструє базові принципи роботи з аналого-цифровим перетворювачем мікроконтролера STM32F407VGT6. Програма зчитує аналоговий сигнал з потенціометра, виконує багаторазове вимірювання (oversampling) для фільтрації шумів та конвертує отримані цифрові значення у вольти. Додатково реалізовано апаратну індикацію рівня напруги за допомогою GPIO (керування світлодіодами).

## Технічні характеристики та налаштування
* **Режим АЦП:** 12-bit Resolution (значення від 0 до 4095).
* **Канал вимірювання:** ADC1_IN3 (пін PA3).
* **Режим зчитування:** Polling mode (програмне опитування).
* **Опорна напруга (Vref):** 3.3 В.
* **Алгоритм підвищення точності:** Програмне усереднення 64 послідовних вимірювань.

## Індикація
Задано порогове значення АЦП = 3000 (~2.41 В):
* Якщо поточне значення `< 3000`: вмикається червоний світлодіод (PD14).
* Якщо поточне значення `>= 3000`: вмикається зелений світлодіод (PD12).
