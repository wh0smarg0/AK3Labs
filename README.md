# Workshop #4: Керування світлодіодами через інтерфейс USART

## 📝 Опис проєкту
Метою даної роботи є реалізація системи віддаленого керування периферією мікроконтролера за допомогою інтерфейсу **USART**. Проєкт побудований за архітектурою **PC Master -> STM32F407 Slave**, де персональний комп'ютер виступає ініціатором команд, а плата виконує роль підлеглого пристрою.

Програма реалізована в режимі опитування (**Polling mode**). Користувач надсилає символи через термінал PuTTY, мікроконтролер обробляє їх, перемикає відповідні світлодіоди та повертає текстовий звіт про виконання команди.

## 🛠 Технічні характеристики
* **Мікроконтролер:** STM32F407VGT6.
* **Інтерфейс:** USART3 (Asynchronous mode).
* **Швидкість передачі:** 19200 baud.
* **Конфігурація кадру:** 8 bits, no parity, 1 stop bit.
* **Hardware Flow Control:** None.
* **Використані піни:**
    * **PD8:** USART3_TX (передача даних на ПК).
    * **PD9:** USART3_RX (прийом даних з ПК).
    * **PD12 - PD15:** Виходи для керування світлодіодами (Green, Orange, Red, Blue).

## ⌨️ Список команд керування
У терміналі PuTTY використовуються наступні клавіші для керування станом плати:

| Клавіша | Дія | Відповідь системи |
| :--- | :--- | :--- |
| **1** | Увімкнути GREEN LED | `Green ON` |
| **2** | Вимкнути GREEN LED | `Green OFF` |
| **3** | Увімкнути ORANGE LED | `Orange ON` |
| **4** | Вимкнути ORANGE LED | `Orange OFF` |
| **5** | Увімкнути RED LED | `Red ON` |
| **6** | Вимкнути RED LED | `Red OFF` |
| **7** | Увімкнути BLUE LED | `Blue ON` |
| **8** | Вимкнути BLUE LED | `Blue OFF` |
| **9** | Увімкнути ВСІ світлодіоди | `All ON` |
| **0** | Вимкнути ВСІ світлодіоди | `All OFF` |

## 🔍 Аналіз сигналів
Для верифікації роботи інтерфейсу було використано логічний аналізатор **Saleae Logic16**.
На знятому трейсі зафіксовано:
1.  **RX (Channel 4):** Отримання від ПК стартового біта та коду символу команди (наприклад, `0x32` для клавіші '2').
2.  **TX (Channel 5):** Відправка мікроконтролером послідовності ASCII-символів текстового повідомлення у відповідь.

## 🚀 Як запустити
1.  Підключити USB-TTL конвертер до відповідних пінів (PD8, PD9, GND).
2.  Відкрити **PuTTY**, обрати тип підключення **Serial**.
3.  Встановити номер COM-порту (згідно з Device Manager) та швидкість **19200**.
4.  Скомпілювати та прошити мікроконтролер через STM32CubeIDE.
5.  Натиснути **Resume (F8)** у дебагері для старту програми.
