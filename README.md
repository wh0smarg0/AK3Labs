# Workshop #2: Апаратний ШІМ (PWM) на STM32

## Опис проєкту
Цей проєкт демонструє використання таймерів загального призначення (TIM4) мікроконтролера **STM32F407VGT6** для апаратної генерації широтно-імпульсної модуляції (ШІМ / PWM). Код не використовує процесорний час (цикл `while(1)` порожній), вся генерація відбувається на апаратному рівні. Ця робота є концептуальною основою для реалізації плавного димування в розробці "розумного освітлення".

## Параметри генерації (Варіант 17)
* **Робоча частота ШІМ:** 4500 Гц (Prescaler = 16, Period = 222)
* **CH1 (Зелений світлодіод, PD12):** Шпаруватість 21%
* **CH2 (Помаранчевий світлодіод, PD13):** Шпаруватість 77%
* **CH3 (Червоний світлодіод, PD14):** Шпаруватість 10%
* **CH4 (Синій світлодіод, PD15):** Шпаруватість 50%

## Апаратне та програмне забезпечення
* **Мікроконтролер:** STM32F407VGT6
* **Інструмент аналізу:** Логічний аналізатор Saleae (Logic 2)
* **IDE:** STM32CubeIDE (C, HAL Framework)
