# Workshop #1: Світлодіодна гірлянда на STM32

## Опис проєкту
Цей проєкт є базовою реалізацією програми на "голому залізі" (Bare Metal) для мікроконтролера **STM32F407VGT6**. Програма реалізує циклічний алгоритм керування чотирма світлодіодами, створюючи ефект "гірлянди". Ця лабораторна робота також слугує першим кроком і базовою апаратною практикою для подальшої розробки курсового проєкту — контролера розумного освітлення.

## Апаратне забезпечення
* Мікроконтролер: **STM32F407VGT6**
* Логічний аналізатор: **Saleae Logic** (для зняття часових діаграм)

## Підключення (Pinout)
Керування здійснюється через порти вводу-виводу загального призначення (GPIO), які налаштовані на цифровий вихід (`GPIO_Output`):
* **PD12** — Зелений світлодіод (Saleae Channel 0)
* **PD13** — Помаранчевий світлодіод (Saleae Channel 1)
* **PD14** — Червоний світлодіод (Saleae Channel 2)
* **PD15** — Синій світлодіод (Saleae Channel 3)

## Алгоритм роботи
Логіка роботи алгоритму є нескінченною і поділена на 3 етапи:
1. **Пряма хвиля:** Послідовне вмикання світлодіодів від зеленого до синього із затримкою 200 мс.
2. **Зворотна хвиля:** Послідовне вимикання світлодіодів у зворотному напрямку (від синього до зеленого) із затримкою 200 мс.
3. **Синхронне мерехтіння:** Трикратне одночасне вмикання та вимикання всіх чотирьох світлодіодів із затримкою 150 мс.
*Між повними циклами гірлянди передбачена пауза 500 мс.*

## Використане програмне забезпечення
* **STM32CubeIDE** — для генерації ініціалізаційного коду, написання логіки мовою С, компіляції та прошивки мікроконтролера через ST-LINK.
* **Saleae Logic 2** — для зняття та аналізу часових діаграм (трейсів) апаратних сигналів з пінів мікроконтролера.
